Welcome to metaphony! Your goal is to identify the best figurative analogy among a selection of ridiculous choices. See how many you can get right before making a mistake! Or upload your own list for a more tailored experience.

Custom Game Setup

To play a custom game, you'll need to create a list of metaphors. Please follow these steps:

  1. Download the blank template file CSV file.
  2. Open the file with a program like Google Sheets or Excel.
  3. Fill in your own good and bad metaphors, one per row.
  4. Export your completed file as a CSV.
  5. Import your new CSV file below and play!
